# encryption-table-py
## Simple Transposition Cipher (11 × 9 Table)
Python implementation of a classic columnar transposition cipher using an 11×9 table.

### Task
Perform encryption/decryption using the cipher table method according to the option. All spaces are taken into account during encryption/decryption. Spaces in the ciphertext are indicated by the “_” symbol.

The condition for my variant:
Encrypt the message “МУЗЫКА ОБЛАДАЕТ МАГИЧЕСКОЙ СИЛОЙ – ВДРУГ
СОБИРАЕТ РАССЕЯННЫЕ МЫСЛИ И ДАЕТ ПОКОЙ ВСТРЕВОЖЕННОЙ ДУШЕ.” by simple transposition method. The size of the cipher table is (11 × 9).

**Input data:**
- `source = "YOUR_TEXT_HERE_WITH_SPACES_REPLACED_BY_"`;
- `n = 11    # number of columns`;
- `m = 9     # number of rows`.
